# Agent Instructions

This repository uses a shared mandatory work log.

After completing any task, every coding agent must add a new entry to the top of:

```text
docs/WORK_LOG.md
```

The entry must include:

- Task summary
- Files or areas changed
- Verification performed
- Notes, risks, or next steps

Before starting work, read:

```text
docs/README.md
docs/handoff/PROJECT_OVERVIEW_AND_REQUIREMENTS.md
docs/WORK_LOG.md
```

Do not commit secrets, `.env` files, database files, coverage output, or generated dependency folders.
