# Bughair
**App:** Fitness Management System

---

## Feature Request 1: Member Profile Edit & Password Recovery System

This feature allows members to update their personal information, such as their **name, phone number, and profile picture**. It helps keep user data correct and up to date.

It also adds a **"Forgot Password"** function, where users can reset their password through their email. The system sends a secure link so users can safely create a new password if they forget the old one.

Overall, this feature makes the system easier to use and more secure, completing the missing parts of the **User Management** system so it works properly in real life.

---

## Feature Request 2: Trainer Rating and Review System

The **Trainer Rating and Review System** allows members to rate trainers (1–5 stars) and leave feedback after completing a private training session. The system displays the average rating and reviews on each trainer’s profile to help other members make better decisions.

Only members who have **booked and completed a session** can submit a review, ensuring fairness and reliability. The system also prevents duplicate reviews and allows administrators to manage inappropriate content.
